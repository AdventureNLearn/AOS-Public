# AOS-Public v0.1.0 Release Notes

**Release Date:** June 6, 2026

## Overview

AOS-Public v0.1.0 is the initial public release of a clean, replicable set of frameworks designed for **civic intelligence, oversight, and truth-seeking work**.

These tools support individuals and small teams in analyzing power structures, breaking through narrative control layers, hardening processes against failure and capture, and producing verifiable, evidence-based findings.

## What's Included

### Core Verification Primitives
- Evidence Gate
- Mission Spine Guard
- Anti-Pattern Scanner

### Key Protocols & Frameworks
- Shatter Protocol (Layer-0 pre-filter for narrative and prompt clarity)
- Sovereign Lens (structured reasoning overlay)
- Influence & Organizational Mapping Framework
- Ops Hardening Framework
- Oversight & Verification Kit Framework

### Operational Support
- Solo Operator Playbook
- Multiple activation fragments and worked examples

### Documentation
- Getting Started guide
- Architecture overview
- Civic-oriented framing throughout

## Philosophy

AOS-Public exists to strengthen independent analysis and public accountability. The frameworks are built with a strong emphasis on Truth-Seeking, resistance to hidden control layers, and practical use in civic and institutional environments.
