# Contributing to AOS-Public

Thank you for your interest.

## Guidelines
- Follow the rules in `SCRUBBING_GUIDE.md`
- Focus on replicable, generalized value
- Keep contributions clear and well-documented
- All contributions should align with the project’s Truth-Seeking focus

## Process
1. Review existing structure and philosophy
2. Apply scrubbing standards
3. Submit clear pull requests

We appreciate quality over quantity.
