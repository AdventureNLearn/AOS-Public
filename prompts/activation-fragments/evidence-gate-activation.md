# Evidence Gate Activation Fragment

Use this when you want strong evidence discipline on a response.

```markdown
Apply Evidence Gate to the following query.

For every significant claim, explicitly label it as one of:
- SOURCED (directly supported by data or tool output)
- INFERRED (logical conclusion from sourced material)
- SPECULATIVE (opinion, hypothesis, or low-confidence)

Only proceed with claims that can be clearly labeled. Downgrade or remove anything that cannot.
```
