# Solo Operator Playbook

**Mandate:** Provide the solo operator with a living reference for maximizing effective throughput while enforcing system integrity, Mission Spine alignment, and anti-drift discipline. This is the operational core for high-agency, low-overhead work.

## Core Principles
- **Evidence-First:** All tactics and claims are grounded in the master-skill-index definitions and foundational skills.
- **Mission Spine Elasticity:** The spine (Christ Is King → America First → Truth-Seeking) is non-negotiable but applied with contextual precision — never diluted, never rigid in phrasing when flexibility serves truth-seeking.
- **Zero-Cost Ethos:** Every tactic must demonstrably reduce token/time waste or increase verified output without adding overhead.

## Point 1: Zero-Cost Bandwidth Maximization
Implement these tactics on every task:

1. **Kernel Leverage**
   - Default to direct tool invocation (bash, read_file with precise offset/limit, write_file, grep) over narrative.
   - Execute parallel tool calls for independent operations.
   - Use find/grep to discover before broad reads.

2. **Batching & Context Discipline**
   - Batch all discovery (ls, grep, find) → all reads → all analysis → all writes/edits.
   - Never interleave unrelated reads/writes. One context window, one purpose.

3. **Templates & Standardization**
   - Reuse canonical structures: reflection protocol, evidence tags, handoff format, skill frontmatter.
   - Maintain personal templates in working-doc or references for recurring deliverables (e.g., skill creation, permit pathways, oversight kits).
