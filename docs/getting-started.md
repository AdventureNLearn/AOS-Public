# Getting Started with AOS-Public

This guide will help you begin using these frameworks for civic intelligence, oversight, and public accountability work.

## What is AOS-Public?

AOS-Public is a collection of clean, replicable frameworks designed for **civic intelligence** — research, oversight, influence mapping, and truth-seeking in the public interest. These tools help individuals and small teams analyze power structures, break narrative control layers, harden processes, and produce verifiable findings.

## Core Philosophy

- **Truth-Seeking First**: Prioritize evidence and resist hidden control or illusion layers.
- **Civic Focus**: Built for oversight, public research, and accountability work.
- **Modular & Replicable**: Use what you need; compose frameworks as needed.
- **OpSec Conscious**: All materials are scrubbed for safe public use.

## Recommended Starting Path

1. Read the main [README.md](../README.md)
2. Read this Getting Started guide
3. Start with the **Shatter Protocol** (`protocols/shatter-protocol.md`)
4. Add **Evidence Gate** for verification discipline
5. Explore **Influence Mapping** and **Oversight Kit** frameworks for deeper civic work

## Quick Start

Use the Shatter Protocol activation fragment on queries involving narrative, power, institutions, or complex reasoning. Follow with evidence labeling for stronger results.
