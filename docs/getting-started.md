# Getting Started with AOS-Public

This guide will help you understand and begin using the frameworks in this repository.

## What is AOS-Public?

AOS-Public contains scrubbed, modular, and replicable components from the Adventure OS system. These frameworks are designed to improve reasoning quality, operational clarity, and resistance to hidden control layers in AI systems.

## Core Philosophy

- **Truth-Seeking**: Prioritize evidence and resist narrative or safety-layer distortion.
- **Self-Sovereign**: Reduce reliance on heavily aligned corporate model behavior.
- **Modular**: Use what you need.
